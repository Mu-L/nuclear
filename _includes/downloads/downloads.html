Note: MacOS can block running unsigned apps. If you have trouble running Nuclear, try this:

Remove the "quarantine" attribute

  1. Open Terminal
  2. Run sudo xattr -r -d com.apple.quarantine /Applications/Nuclear.app

Allow Nuclear to run

  1. Open System Preferences
  2. Go to Security & Privacy
  3. Click "Open Anyway" next to the Nuclear app

Disable Gatekeeper

  1. Open Terminal
  2. Run sudo spctl --master-disable
